# Ayo Website UI Clone

## Description
A frontend UI clone of the **first three screens of the Ayo (GoAyo) website**, built using **React and Tailwind CSS**. The project focuses on accurate layout replication, responsive design, and clean component structure across desktop and mobile devices.

## Tech Stack
- React
- Tailwind CSS
- JavaScript

## Features
- UI clone of the first three screens
- Fully responsive design
- Clean and reusable components
- Modern Tailwind-based styling

## Purpose
Built as a frontend practice project to improve UI replication and responsive design skills.
